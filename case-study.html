Case Study

Sid Perfume Case Study

A deep dive into the design and development of a luxury fragrance e-commerce experience.

The Challenge

Create a high-performance, visually stunning e-commerce experience for a luxury perfume brand that conveys exclusivity, craftsmanship, and prestige — while ensuring instant load times and buttery-smooth animations.

The Approach

We designed a dark, sophisticated palette featuring midnight blue, champagne gold, and emerald accents. Glassmorphism cards, custom cursor interactions, and GPU-accelerated CSS animations deliver a tactile, immersive feel. The welcome page loads instantly with zero external dependencies.

Key Features

  • Zero-dependency welcome page — loads in under 150ms
  • CSS-only morphing clip-path background animations
  • Staggered text reveals with hardware-accelerated transforms
  • Full glassmorphism design system with mouse-tracking tilt
  • Responsive across all viewports (mobile → 4K)
  • Custom cursor with smooth ring physics

Tech Stack

HTML5 CSS3 Vanilla JS CSS Animations Glassmorphism
